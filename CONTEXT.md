# Dymasco Apriso CSS Training

Glossaire du kit de formation CSS Apriso Dymasco. Il fixe les termes pédagogiques utilisés dans les supports.

## Language

**Fiche apprenant**:
Support distribué aux participants. Elle contient la demande client, les contraintes, les paliers et les livrables, sans corrigé ni liste de sélecteurs à modifier.
_Avoid_: fiche exercice complète, corrigé étudiant

**Fiche formateur**:
Support interne d'animation. Elle contient les pièges attendus, la trajectoire de correction, les sélecteurs probables et les critères de débrief.
_Avoid_: fiche apprenant enrichie

**Palier**:
Niveau de réussite observable dans l'exercice final. Il permet de corriger sans logique tout-ou-rien.
_Avoid_: note, score

**Demande client**:
Formulation métier qui justifie les choix CSS. Elle prime sur une liste de propriétés techniques à appliquer.
_Avoid_: brief technique

**Dette Process Builder**:
Écart où la meilleure correction devrait être faite dans Apriso / Process Builder plutôt qu'en CSS. Elle peut être contournée dans le kit, mais doit être nommée.
_Avoid_: bug CSS
